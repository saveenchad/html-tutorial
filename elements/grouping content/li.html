Definition:

The <li> tag defines a list item. The <li> tag is used in ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).

Examples:

Example 1:

This is an example of an ordered HTML list:

Name of students:

  1. Sara
  2. Saveen
  3. Stephen
  4. Joe
  5. Vlad

Example 2:

This is an example of an unordered HTML list:

Name of students:

  • Sara
  • Saveen
  • Stephen
  • Joe
  • Vlad
