Definition:

An element that allows audio files to be played on a web page

Usage:

<audio autoplay controls loop muted preload="auto | metadata | none" src="URL">
  <source src=".mp3 | .ogg | .wav" type="audio/mp3 | audio/ogg | audio/wav">
  ...
</audio>

Examples:

This example was originally found on the W3Schools tutorial for the audio tag

<audio controls>
  <source src="horse.mp3" type="audio/mp3">
</audio>

Note: the audio content would play if it existed...
