Definition:

Scripts are small programs that can be embedded using the <script> element. It defines a client-side script.

Examples:

Example 1:

This example is on w3:
It is a form with a script that sets the value of the form's <output> element to say "Hello World".

The code looks like:

<form name="main">

Result: <output name="result"></output>

<script>

document.forms.main.elements.result.value = 'This is coming from the <script> tag.'

</script>

</form>
