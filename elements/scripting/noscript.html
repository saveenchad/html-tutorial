Definition:

The <noscript> tag is used to provide an alternate content for users that have disabled scripts in their browser or have a browser that doesn't support client-side scripting. It defines an alternate content for users that do not support client-side scripts. The content inside the <noscript> element will only be displayed if scripts are not supported, or are disabled in the user's browser:

Examples:

Example 1:

The code looks like:

<p id="demo">Original message</p>

<script>

document.getElementById("demo").innerHTML = 'This is the message in <script> tag.'

<script>

<noscript>This is the message with <noscript> tag.</noscript>
