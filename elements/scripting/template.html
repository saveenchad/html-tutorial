Definition:

Allows repeated HTML to be condensed into a template that can be appended to another element via JS

Usage:

First, parent HTML is required so that the template can be appended to it

<ol id="myList">
</ol>

Next, make a template of the item that will be appended to the parent HTML

<template id="myListItemTemplate">
  <li class="myListItem"></li>
</template>

Finally, add some JS to dynamically append your template to your parent HTML

<script>
  //grab an instance of the template
  var temp = document.querySelector('#myListItemTemplate');
  var tempDate = temp.content.querySelectorAll("li");
  tempDate[0].textContent = "Template list item";

  //clone the template and add it to the parent HTML
  var list = document.getElementById("myList");
  var newItem = document.importNode(temp.content, true);
  list[0].appendChild(newItem);
  ...
</script>
