Definition:

The HTML <canvas> element is used to draw graphics, on the fly, via scripting (usually JavaScript). The <canvas> element is only a container for graphics. You must use a script to actually draw the graphics. Canvas has several methods for drawing paths, boxes, circles, text, and adding images.

Examples:

Example 1:

Draw an empty rectangle:

The code looks like:

<canvas id="myCanvas" width="400" height="50" style="border:10px solid #000000;"> Your browser does not support the HTML5 canvas tag. </canvas>

The results look like:

Your browser does not support the HTML5 canvas tag.

Example 2:

Draw a circle using JavaScript:

The code looks like:

<canvas id="myCanvas2" width="200" height="100" style="border:10px solid #A3F3D3;"> Your browser does not support the HTML5 canvas tag. </canvas>

<script>

var c = document.getElementById("myCanvas2");
var ctx = c.getContext("2d");
ctx.beginPath();
ctx.arc(95,50,40,0,2*Math.PI);
ctx.stroke();

</script>

The results look like:

Your browser does not support the HTML5 canvas tag.

Example 3:

Draw a stroke text:

The code looks like:

<canvas id="myCanvas3" width="270" height="80" style="border:5px solid #d3d3d3;"> Your browser does not support the HTML5 canvas tag. </canvas>

<script>

var c = document.getElementById("myCanvas3");
var ctx = c.getContext("2d");
ctx.font = "45px Times New Roman";
ctx.strokeText("Sara's Canvas",10,50);

</script>

The results look like:

Your browser does not support the HTML5 canvas tag.
