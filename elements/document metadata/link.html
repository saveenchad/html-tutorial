Definition:

The <link> tag defines a link between a document and an external resource.
The <link> tag is used to link to external style sheets.

Examples:

Example 1:

h2 tag's color has changed.

The code looks like:

<link href="style.css" rel="stylesheet">

Example 2:

If you hover on the back button on top of the screen, it will turn red. That effect comes from the link tag.
