Definition:

Attributes are placed in the start tag and consist of a name and a value separated by an "=" character. They provide additional information about the element such as identity, style, and data.

Usage:

<element attribute1="value1" attribute2="value2" ... ></element>

Examples:

Here is an anchor element with an href attribute which allows it to link to other parts of this document or other files

<a href="#"></a>

The style attribute allows CSS rules to be applied to an element

<p style="color:red">I will be red!</p>


The id and class attributes allow an element to be accessed via CSS and JavaScript

<div id="myID" class="myClass"></div>
