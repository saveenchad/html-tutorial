HTML includes rules that allow only certain elements to be nested within other elements.

This prevents a web developer from running wild with their tags

For example:

<ol>
  <ol>
    <li>Nested list element</li>
  </ol>
  <li>Normal list element 1</li>
  <li>Normal list element 2</li>
</ol>

This is disallowed because lists should only have list items within it.
