Definition:

An attribute that essentially sets strict mode for the html document. It won't load unless all rules are followed!

Usage:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
  ...
</html>

The following rules need to be followed in order to be rendered properly:

  1. The xhtml doctype is mandatory.
  2. The xmlns attribute in the html tag is mandatory
  3. The following elements are mandatory: html, head, title, and body
  4. All elements are properly nested
  5. All start tags must have matching end tags
  6. All start and end tags must be in lower case
  7. All documents must have only one root element
  8. All attributes must be in lower case
  9. All attribute values must be quoted
  10. Any form of attribute minimization is forbidden
