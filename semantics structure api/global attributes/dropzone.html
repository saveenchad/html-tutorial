Definition:

An attribute that determines the result of dragging and dropping an element

Usage:

THIS ATTRIBUTE IS NOT SUPPORTED IN ANY WIDELY USED BROWSER!

<element dropzone="copy | move | link"><element>

Examples:

<p dropzone="copy">Dropping me makes a copy of the dragged element at the drop location</p>

Dropping me makes a copy of the dragged element at the drop location


<p dropzone="move">Dropping me moves the dragged element to the drop location</p>

Dropping me moves the dragged element to the drop location


<p dropzone="link">Dropping me makes a link of the original element at the drop location</p>

Dropping me makes a link of the original element at the drop location
