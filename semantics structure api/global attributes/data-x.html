Definition:

An attribute that allows data to be stored by the programmer in any element

Usage:

<element data-insert-keyword-here-that-represents-what-youre-storing="value-to-store"></element>

Examples:

Given the following JS function:

function calcArea(rect) {
  var x = rect.getAttribute("data-width");
  var y = rect.getAttribute("data-length");
  alert("the area of this rectangle is " + (x * y) + " px^2");
}

<div data-width="100" data-length="300" onclick="calcArea(this)">100 x 300</div>
<div data-width="200" data-length="200" onclick="calcArea(this)">200 x 200</div>
<div data-width="200" data-length="300" onclick="calcArea(this)">200 x 300</div>
<div data-width="700" data-length="200" onclick="calcArea(this)">700 x 200</div>
