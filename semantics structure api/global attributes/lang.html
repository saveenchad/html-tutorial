Definition:

Un attribut qui definit la langue principale (BCP 47) pour le contenu de l'element. Cela se fait generalement dans la balise html pour l'ensemble du document.

Usage:

<element lang="la langue | "></element>

Exemples:

<html lang="fr">
  ...
</html>

<p>Ce document a ete redige en francais!</p>

Ce document a ete redige en francais!


<p>mais depuis le html lang est "fr", la page peut etre traduit en anglais!</p>
