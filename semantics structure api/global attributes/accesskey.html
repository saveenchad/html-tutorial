Definition:

An attribute that sets a keyboard shortcut to focus a focusable element

Usage:

<focusable-element accesskey="character"></focusable-element>

Examples:

<input type="text" accesskey="k"/>

Hit alt + k (chrome) or alt + shift + k (firefox) to focus the textbox below!


<a href="https://www.google.com" accesskey="q">Go to Google!</a>

Hit alt + q (chrome) or alt + shift + q (firefox) to use the link below!

Go to Google!
