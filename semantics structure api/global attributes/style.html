Definition:

An attribute that can set CSS rules to elements via HTML

Usage:

Note: This is generally not the accepted way of applying CSS rules as it is much cleaner to add a style tag in the head section and add CSS rules there. However, doing so will decrease the readability of your HTML. 

<element style="css-property-1:css-value-1; css-property-2:css-value-2"></element>

Examples:

<p style="background:black;color:white;font-size:20px;border-radius:50px;text-align:center">I have a bunch of CSS applied to me!</p>
