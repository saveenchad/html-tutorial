Definition:

An attribute that allows a set of elements to be selected via CSS and JS selectors

Usage:

<element class="token"></element>

Examples:

Given the following CSS:

.big-font { font-size: 30px; }

.blue { color: blue; }

And the following JS:

function changeColor() {

  var el = document.getElementsByClassName("blue")[0];

  el.style.color = "red";

  el.innerHTML = "I'm red now!";

}


<p class="big-font">I'm HUUUGE!</p>

I'm HUUUGE!

<p class="blue">I'm blue!</p>

I'm blue!
