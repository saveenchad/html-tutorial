Definition:

Un attribut qui definit la langue principale (BCP 47) pour le contenu de l'element. Cela se fait generalement dans la balise html pour l'ensemble du document.

Usage:

THIS ATTRIBUTE IS NOT SUPPORTED BY ANY WIDELY USED BROWSERS!

<element traduire="oui | non"></element>

Exemples:

<html lang="fr">
  ...
</html>

<p translate="no">Ce document a ete redige en francais, mais vous ne pouvez pas traduire, donc vous ne saurez pas...</p>
