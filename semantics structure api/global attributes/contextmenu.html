Definition:

An attribute that gives an element a context menu by tying it with a menu element

Usage:

THIS ATTRIBUTE IS ONLY SUPPORTED IN FIREFOX!

<element contextmenu="unique-menu-name"><element>

<menu type="context" id="unique-menu-name">

  <menuitem ... >

  ...

</menu>

Examples:

<p contextmenu="p-menu">I have a context menu!</p>

<menu type="context" id="p-menu">

  <menuitem label="my custom option">

</menu>
