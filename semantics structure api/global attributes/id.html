Definition:

An attribute that allows a specific element to be selected via CSS or JS

Usage:

<element id="token"></element>

Examples:

Given the following CSS:

#tiny-font { font-size: 8px; }

#red { color: red; }

And the following JS:

function changeColor() {

  var el = document.getElementById("red");

  el.style.color = "green";

  el.innerHTML = "I'm green now!";

}


<p id="tiny-font">I'm tiiiny!</p>

I'm tiiiny!

<p id="red">I'm red!</p>

I'm red!
