Definition:

An attribute that sets the order in which a set of elements get focus when the tab button is pressed by the user

Usage:

<element tabindex="0 | 1 | 2 | ..."></element>

Examples:

Use the tab button to cycle through the following form

<form>
  <input type="text" placeholder="First Name" tabindex="3" />
  <input type="text" placeholder="Last Name" tabindex="2" />
  <input type="email" placeholder="Email" tabindex="1" />
</form>
